# To Do List App "TASK MANAGEMENT"

--> This is only a "demonstrator" with no backend.

Built with:
- HTML
- Bootstrap
- CSS
- JavaScript
using Local Storage

This app fulfills the following stories:
As a user i can
- log in into the task manager
- add a task
- read a list of tasks
- search for a task
- edit any task
- mark any task as "completed"
- delete any task
- delete all task
- log out
